PushNetworkDetector Klasse
Ermöglicht das auffinden von Servern im lokalen Netzwerk. Dieser NetworkDetctor muss die Suche manuell starten. Hiefür muss die Refresh Methode aufgerufen werden. Ohne diesen Aufruf wird die Liste NICHT automatisch aktualisert. Beim Aufruf von StartDetection wird automatisch erstmalig Refresh aufgerufen.
